# 🐝 Złota Pasieka (The Golden Apiary)

Modern, static, multilingual website for an educational apiary and beekeeping dashboard.

## 🚀 Live Demo
Managed via GitHub Pages at: [https://czterylapynawakacjach.github.io/zlota-pasieka/](https://czterylapynawakacjach.github.io/zlota-pasieka/)

## 🛠 Features
- **Bee-Ops Dashboard:** Real-time beekeeping telemetry (IMGW-PIB).
- **Daily Buzz:** Automated news aggregator from global and local sources.
- **Seasonal Insights:** GDD tracking and Nectar Flow (Delta T) indicators.
- **100% Static:** High performance, zero runtime API overhead.
